Creating a repository in GitHub

This section describes how to create a repository in GitHub.

  1. In the address field of your internet browser, type the following URL address: https://github.com/login.
  2. Type the username and password of your GitHub account, and click Sign in.
  3. Click New in the left pane. The Create a new repository page displays.
  4. Type a new repository name in the Repository name field, for example, hupo6666.github.io.
  5. Select Public checkbox.
  6. Click Create repository. The new repository is created.
