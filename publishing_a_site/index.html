Publishing a site

This section describes how to publish a site by GitHub.

  1. In Windows 10, type Command Prompt in the search box, and click Command Prompt from the search result. The Command Prompt console appears.

  2. In Command Prompt, enter the following command:

    cd c:\hugo

  3. Enter the following command:

    hugo -t <local theme name>

    For example:

    hugo -t hugo-theme-techdoc

    Hugo generates output files to the public folder.

  4. Modify the value of baseUrl attribute to https:// in the config.toml file under the site name, for example, baseURL = “https://hupo6666.github.io”.

  5. Access the public folder:

    cd <public folder path>

    For example:

    cd c:\hugo\my_documentation

  6. Enter the following commands:

    git init 
git add . 
git commit -m "Initial commit." 
git remote add origin https://github.com/hupo6666/hupo6666.github.io.git 
git push -u origin master
