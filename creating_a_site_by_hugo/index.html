Creating a local documentation site by Hugo

This section describes how to create a local document site by Hugo.

  1. In Windows 10, type Command Prompt in the search box, and click Command Prompt from the search result. The Command Prompt console appears.

  2. In Command Prompt, enter the following command:

    cd c:\hugo

  3. Generate a new site.

    hugo new site my_documentation

  4. Download a documentation theme, for example, Techdoc theme.

    cd <new site name>
git clone <Web URL of a theme> themes/<local theme name>

    For example:

    cd my_documentation
git clone https://github.com/thingsym/hugo-theme-techdoc.git themes/hugo-theme-techdoc

  5. Copy the MD files of documents to the content folder of the local site, for example, C:\Hugo\my_documentation\content. The directory structure of MD files in the content folder is as follows:

    content
  |__ _index.md
  |__- creating_a_site_by_hugo
      - _index.md
  |__ creating_a_respository_in_github
      - _index.md
  |__ publishing_a_site
      - _index.md

  6. Start the Hugo server.

    hugo server --theme=<local theme name>

    For example:

    hugo server --theme=hugo-theme-techdoc

  7. In the address field of your internet browser, type the following URL address: http://localhost:1313. The local site opens.
